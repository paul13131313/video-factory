Video Factory

AIチャットの会話を、そのまま動画に変換。
テキストを入力するだけで、リアルなチャット風MP4が完成します。

Claude ChatGPT LINE
セットアップ
  1. 1
    リポジトリをクローン
    git clone https://github.com/paul13131313/video-factory.git
  2. 2
    依存パッケージをインストール
    cd video-factory && npm install
    ※ Node.js 18+ と FFmpeg が必要です
  3. 3
    起動
    node server.js
    ブラウザで http://localhost:3000 を開いて使えます。
